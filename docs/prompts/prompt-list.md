## Quality Standards

Every prompt includes:

- **TASK**: Clear, actionable objective
- **SYSTEM CONTEXT**: Why it exists, what problem it solves
- **REQUIREMENTS**: Detailed implementation steps (NO CODE)
- **CONSTRAINTS**: Hard limits, performance requirements
- **INTEGRATION POINTS**: Dependencies and future connections
- **TESTING REQUIREMENTS**: Specific test cases
- **ACCEPTANCE CRITERIA**: Measurable success metrics
- **QUALITY STANDARDS**: Best practices to follow
- **DELIVERABLES**: Concrete outputs expected
- **SUCCESS METRIC**: One-line definition of success
